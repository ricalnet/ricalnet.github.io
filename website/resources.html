Self-Hosted Infrastructure

Bangun Infrastruktur Mandiri

Bangun Infrastruktur Berdaulat. Kedaulatan teknologi dimulai dari server sendiri. Kurangi ketergantungan pada cloud asing

Private Cloud (Nextcloud)

Ganti Google Drive, iCloud, atau OneDrive dengan cloud pribadi. File, kontak, kalender — semuanya di server Anda sendiri. Data ownership penuh (open source, dikelola sendiri).

Self-Hosted Kendali Penuh

Password Manager (Vaultwarden)

Hosting password manager sendiri. Alternatif open source untuk LastPass/Bitwarden dengan server pribadi. Kredensial Anda hanya dapat diakses oleh Anda (dengan asumsi keamanan server terjaga).

Self-Hosted Terenkripsi

Media Server (Jellyfin)

Streaming film, musik, dan foto dari server rumah Anda. Alternatif open source untuk Netflix/Spotify yang tidak melacak aktivitas menonton Anda.

Peralatan Kedaulatan Digital

Dari browser hingga server — bangun tumpukan teknologi yang lebih Anda kendalikan

Tor Browser

torproject.org
Browser Anonimitas

Browsing dengan anonimitas yang baik. Lalu lintas melalui 3 relay Tor. Mengurangi pelacakan dan membuka akses di wilayah dengan sensor.

Open Source, Gecko

LibreWolf

librewolf.net
Browser Privacy-Focused

Firefox yang dikuatkan untuk privasi. Telemetri dimatikan, fingerprinting dilindungi, dan iklan diblokir secara default.

Open Source, Gecko

Brave Browser

brave.com
Browser Ad Blocker Bawaan

Browser berbasis Chromium dengan perlindungan privasi bawaan. Blokir iklan & tracker tanpa ekstensi tambahan.

Open Source, Chromium

QubesOS

qubes-os.org
Operating System Security by Isolation

OS yang mengisolasi aplikasi dalam VM terpisah. Standar emas untuk keamanan dan kedaulatan digital tingkat lanjut.

Open Source, Xen-based

TailsOS

tails.net
Operating System Live OS

OS amnesia yang berjalan dari USB. Semua lalu lintas via Tor. Tidak meninggalkan jejak — matikan, lupakan.

Open Source, Tor-based

Debian

debian.org
Operating System Self-Hosted Server

OS server pilihan untuk self-hosting. Stabil, aman, dan didukung ribuan package open source. Fondasi infrastruktur mandiri Anda.

Open Source, Linux

GrapheneOS

grapheneos.org
Operating System Android

OS Android yang di-hardening untuk privasi. Hapus semua layanan Google. Kontrol penuh atas data di ponsel Anda.

Open Source, AOSP-based

/e/OS

e.foundation/e-os/
Operating System Android

Android tanpa Google. Dilengkapi MicroG dan layanan cloud alternatif. Alternatif ramah pengguna untuk de-Google.

Open Source, Privacy-focused

MicroG

microg.org
Android Google Services

Implementasi open source layanan Google Play. Jalankan aplikasi yang butuh GMS tanpa melacak aktivitas Anda.

Open Source, Privacy

F-Droid

f-droid.org
Android App Store

Toko aplikasi open source untuk Android. Semua aplikasi diverifikasi — bebas tracker, bebas ikian.

Open Source, FOSS-only

Aurora Store

auroraoss.com
Android Anonymous Play Store

Klien Google Play Store anonim. Unduh aplikasi dari Play Store tanpa login atau melacak identitas Anda.

Open Source, Anonymous

SearXNG

docs.searxng.org
Search Engine Self-Hosted

Hosting mesin pencari sendiri. Aggregator dari puluhan search engine — dengan hasil yang lebih netral dan privasi yang lebih terjaga.

Open Source, Self-hostable

DuckDuckGo

duckduckgo.com
Search Engine No Tracking

Mesin pencari populer yang tidak melacak. Alternatif siap pakai sebelum beralih ke self-hosted.

Proprietary, Privacy-focused

Briar

briarproject.org
Messaging P2P + Tor

Peer-to-peer messaging via Tor. Tidak ada server pusat — komunikasi Anda hanya antara perangkat.

Open Source, P2P

Session

getsession.org
Messaging No Phone Number

Pesan anonim tanpa nomor telepon. Jaringan onion routing untuk metadata protection.

Open Source, Decentralized

Standard Notes

standardnotes.com
Notes E2EE

Catatan terenkripsi end-to-end. Bisa self-hosted server sinkronisasi sendiri.

Open Source, Cross-platform

Nextcloud

nextcloud.com
Cloud Storage Self-Hosted

Alternatif cloud seperti Google Drive. Host di server pribadi. File, kontak, kalender — semua di kendali Anda.

Open Source, Self-hosted

Orbot

orbot.app
VPN Tor for Android

VPN gratis via Tor network. Anonimkan semua traffic dari ponsel Anda.

Open Source, Tor-based

Vaultwarden

vaultwarden
Password Manager Self-Hosted

Hosting password manager sendiri. Alternatif ringan untuk Bitwarden. Semua kredensial di server Anda.

Open Source, Self-hostable

KeePassXC

keepassxc.org
Password Manager Offline First

Database kata sandi lokal. Tidak ada cloud, tidak ada server. Anda yang memegang file kunci sepenuhnya.

Open Source, Offline

Kedaulatan teknologi bukan tentang satu tools, tapi tentang tumpukan (stack) yang lebih Anda kendalikan.
Mulai dari browser → OS → komunikasi → cloud → semuanya.
